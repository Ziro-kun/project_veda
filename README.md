# Project Veda 🤖🛡️

**Project Veda** 는 건프라(건담 프라모델) 입문자와 빌더들을 위한 AI 시각 보조 및 전술 정보 브리핑 시스템입니다. 사용자가 건프라 이미지를 업로드하면 기체를 식별하고, 해당 기체에 대한 도감 정보 및 브리핑을 제공합니다. AI 엔지니어를 지망하는 사람이 건프라 매니아인 배우자를 위해 만들었습니다.

## 📌 현재 버전 상태 (Working Version)

본 저장소에 업로드된 버전은  **현재 개발 및 작업 중인 기능 구현 버전(`app.py`)** 입니다.

### ✨ 주요 기능

- **이미지 업로드 및 스캔**: 사용자가 촬영하거나 저장한 건프라 이미지를 업로드 (`jpg`, `png`, `jpeg` 지원)
- **전술 데이터 브리핑**:
  - **식별 코드**: 기체명 및 예상 등급 (MG, HG, RG 등) 식별
  - **전술 데이터**: 기체의 소속, 대표 파일럿, 핵심 특징 요약
  - **상세 브리핑**: 애니메이션 내 활약상 및 기체 설정 제공
- **VLM 기반 분석**: 현재 버전은 Google Gemini (Flash / Pro Vision) API를 임시로 연동하여 이미지 분석을 수행하고 있습니다.

### 📂 주요 파일 구성

- **`app.py`**: 메인 웹 애플리케이션입니다. Streamlit을 활용한 UI 구성, 사용자 이미지 입력 처리, 그리고 Google Gemini API를 연동하여 건프라 이미지를 분석하고 전술 데이터를 브리핑하는 핵심 로직이 구현되어 있습니다.
- **`check.py`**: 환경 변수로 설정된 API 키가 정상적으로 동작하는지 확인하고, 현재 사용 가능한 Gemini 모델 목록(특히 비전 모델 지원 여부)을 점검하는 유틸리티 스크립트입니다.

## 🛠️ 설치 및 실행 방법

1. **저장소 클론 및 패키지 설치**

   ```bash
   git clone <repository-url>
   cd project_veda
   pip install streamlit google-generativeai python-dotenv pillow
   ```

2. **환경 변수 설정**
   - 프로젝트 루트 폴더에 `.env` 파일을 생성하고 아래와 같이 Google Gemini API 키를 입력합니다.

   ```env
   GOOGLE_API_KEY="본인의_API_키_입력"
   ```

3. **앱 무구동**
   ```bash
   streamlit run app.py
   ```
